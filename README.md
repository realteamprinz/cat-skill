## What is cat.skill?

A species-specific extension of paw.skill that adds deep feline intelligence — territory mapping, independence pattern analysis, hunting behavior tracking, multi-cat social hierarchy, and a decoder for everything your cat does that makes no sense to humans.

paw.skill knows your pet. cat.skill knows your CAT.

## Install

Requires [paw.skill](https://github.com/realteamprinz/paw-skill) as parent.

```bash
# OpenClaw
clawhub install realteamprinz/paw
clawhub install realteamprinz/cat

# Hermes
hermes skills install paw-skill
hermes skills install cat-skill
```
